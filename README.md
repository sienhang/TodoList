# TodoList

## 一、运行

1. npm install
2. node entry.js
3. http://localhost:3002

有些时候国内npm源访问比较慢，那么推荐使用cnpm。

1. npm install cnpm -g
2. cnpm install
3. node entry.js
4. http://localhost:3002

## 二、主要功能

1. 日历功能
2. 编辑指定日期的todolist
3. 可将未完成项标志为已完成，将已完成项标志为未完成
4. 每天的编辑项将被保存


## 三、主要技术

1. 前台：react、scss、jquery 、webpack、es6、babel
2. 后台：koa2、mongoose、es7   
3. 数据库：mongodb
